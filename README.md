# Veyra — Dark Fantasy RPG

Браузерная dark fantasy RPG, разработанная для экосистемы [Shelby](https://shelby.xyz).

## Играть

**[spr1ntray.github.io/Veyra](https://spr1ntray.github.io/Veyra)**

## Что есть в игре

- **Тренировочная площадка** — бои с манекеном (5 в день), 3 заклинания мага
- **Ежедневные награды** — 7-дневный цикл нарастающих наград
- **Карта мира** — изометрический вид, туман войны над закрытыми локациями
- **Инвентарь** — посох, шляпа, накидка; снаряжение влияет на урон
- **Прогрессия** — уровни 1–10, XP за победы

## Технологии

- Чистый HTML5 / CSS3 / Vanilla JavaScript (ES6 Modules)
- Без фреймворков, без сборщиков
- Хранение прогресса: `localStorage`

## Built on Shelby

Проект создан в рамках программы разработчиков [Shelby Protocol](https://docs.shelby.xyz/protocol) — высокопроизводительной децентрализованной системы хранения данных на блокчейне Aptos.
