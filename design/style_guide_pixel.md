# VEYRA — PIXEL ART STYLE GUIDE
## Минимализм + Современный пиксельный арт

---

## ВИЗУАЛЬНЫЙ СТИЛЬ

### Философия дизайна
Veyra использует **современный pixel art** (16-32 бит эпоха, но не ретро подражание) с **минималистичным подходом**. Каждый пиксель имеет смысл. Пространство используется уважительно — много атмосферного пустого пространства. Фокус на чистоте форм и яркости через контраст, а не через сложность.

### Основной стиль рендеринга
- **Pixel art, 16-32bit era inspired** — острые края, чёткие силуэты, никакой размытости
- **Внутри пикселей используются красивые градиенты** — это не плоский 8-bit, а живой, объёмный pixel art
- **Минимализм** — отказ от лишних деталей, максимум атмосферы при минимуме визуального шума
- **Целеустремлённый контраст** — тёмный фон + яркие акценты создают фокус внимания
- **Слегка готический мрак** — dark fantasy атмосфера, но через простоту и пространство, а не через детальность

### Настроение и атмосфера
- Одинокость и величие (маг в пустом городе)
- Магическое напряжение (неоновые всполохи в тьме)
- Ощущение скрытой силы (мало движения, но много потенциала)

## ЦВЕТОВАЯ ПАЛИТРА

### Первичная тёмная палитра (фоны и основа)
| Цвет | Hex | Применение |
|------|-----|-----------|
| Глубокий чёрный | `#0a0a0e` | Самые тёмные тени, контуры |
| Угольный | `#1a1a24` | Основной тёмный фон |
| Тёмный сланец | `#2a2a38` | Вторичные фоны, вспомогательные элементы |
| Серый тень | `#3a3a48` | Лёгкие тени, разделители |

### Неоновые акценты (магия, фокус внимания)
| Цвет | Hex | Применение |
|------|-----|-----------|
| Неоновый голубой | `#00d9ff` | Магия, заклинания (основной неон) |
| Неоновый фиолетовый | `#b700ff` | Тёмная магия, альтернативные заклинания |
| Неоновый зелёный | `#00ff41` | Здоровье, баффы, позитивные эффекты |
| Неоновый оранжевый | `#ff6b00` | Огонь, опасность, критические эффекты |

### Дополнительные цвета (UI, персонажи, детали)
| Цвет | Hex | Применение |
|------|-----|-----------|
| Ледяной белый | `#e8f0ff` | Текст, светлые детали, магический свет |
| Мягкий серый | `#a0a0b0` | Вторичный текст, неактивные элементы |
| Бронза/Золото | `#c9a76f` | Редкие предметы, драгоценности |
| Кровь-красный | `#8b0000` | Урон, враги, опасность |

### Правила применения цвета
1. Каждый экран использует **максимум 2 неоновых акцента** из списка выше
2. 80% визуала — тёмная палитра, 20% — неоны и светлые элементы
3. Градиенты внутри пиксельных форм: от тёмного оттенка к светлому (объём без деталей)
4. Никогда не используй яркие цвета просто так — только для привлечения внимания к важным элементам

## ТИПОГРАФИКА

### Основной шрифт UI
- **Шрифт**: Monospace или пиксельный шрифт (VT323, Press Start 2P, но не в полный размер)
- **Стиль**: Четкий, читаемый, минималистичный
- **Цвет**: `#e8f0ff` (ледяной белый) для основного текста
- **Вторичный текст**: `#a0a0b0` (мягкий серый)
- **Заголовки**: `#00d9ff` (неоновый голубой) или `#b700ff` (неоновый фиолетовый)

### Размеры текста (условные пиксели)
- **Заголовки экранов**: 32px, жирный, неоновый цвет
- **Основной текст**: 16px, светлый оттенок
- **Мелкий текст**: 12px, серый
- **Числа (урон, HP)**: 20px, яркие цвета (оранжевый урон, зелёный хил)

## ПРАВИЛА ДЛЯ КАЖДОГО ТИПА АССЕТА

### BACKGROUNDS (загрузка, площадь, дом, карта)
- **Размер**: 1920×1080 или адаптивный
- **Стиль**: Pixel art, но с эффектом дистанционирования (дальние планы светлее, ближние контрастнее)
- **Минимализм**: Максимум пустого пространства, минимум деталей
- **Палитра**: Строго тёмная палитра (из первичной) + 1-2 неоновых акцента (например, свечение вдали)
- **Градиенты**: Используй градиенты внутри пиксельных блоков для объёма
- **Композиция**: Вертикальная или панорамная, с сильным центром внимания

### CHARACTERS (спрайты — маг, манекен)
- **Размер**: 256×256 или 512×512 (pixel art с хорошей детализацией)
- **Стиль**: Pixel art спрайт, анимационная поза (динамичная позиция)
- **Палитра персонажа**: Тёмная одежда (уголь, сланец) + 1 неоновый акцент (например, голубой посох)
- **Контраст**: Чистые линии силуэта, внутренние градиенты
- **Фон**: Прозрачный (PNG) или однотонный тёмный фон
- **Ощущение**: Магическое напряжение, даже в статичной позе

### ICONS (заклинания, 64×64)
- **Размер**: Именно 64×64 или 128×128
- **Стиль**: Pixel art, минимальная геометрия (круги, квадраты, линии)
- **Палитра**: Каждый спелл имеет свой неон: голубой (Arcane Bolt), фиолетовый (Focus), зелёный (Shadow Pulse)
- **Фон**: Чёрный или едва видимый градиент из тёмной палитры
- **Негативное пространство**: Много пустоты внутри иконки, фокус на силуэте
- **Контрастность**: Проверь читаемость на 64×64 — каждый пиксель важен

### UI ЭЛЕМЕНТЫ (рамки, панели, кнопки)
- **Стиль**: Пиксельные прямоугольники с четкими границами
- **Палитра**: Серые границы (`#3a3a48`), чёрный фон (`#0a0a0e`)
- **Неоновые акценты**: Только для активных/выделенных элементов (линия неона вокруг кнопки)
- **Минимализм**: Никаких закруглений, украшений — только нужные линии

## ЗАПРЕЩЁННЫЕ ЭЛЕМЕНТЫ

❌ **Никогда не используй:**
- Фотографизм, реалистичные текстуры
- Размытость, мягкие края (soft focus)
- Яркие, кислотные цвета без причины
- Слишком много деталей в фонах (мусор, текстуры, всё подряд)
- Телесные тона (это фэнтези, не реализм)
- Глосс, шайн, отражения (если они не магические эффекты)
- Gradients без чёткого направления (должны выглядеть как pixel art, не как Photoshop)

## ПРИМЕРЫ ПРИМЕНЕНИЯ ПО ЭКРАНАМ

### 1. LOADING SCREEN
- Центральный образ: замок или портал в полутьме
- Цвета: тёмная палитра + голубой неон (портал светится)
- Ощущение: величие, предчувствие приключения
- Минимализм: замок занимает 30% экрана, 70% — пустота и атмосфера

### 2. SQUARE (ПЛОЩАДЬ)
- Вид: булыжная площадь, немного зданий по краям (силуэты)
- Цвета: угольный и сланец (фон) + оранжевые неоновые фонари (уличные огни)
- Минимализм: мало деталей, много пустого пространства для взгляда игрока
- Манекен находится в центре, хорошо видно

### 3. HOME (ДОМ МАГА)
- Вид: башня, комната с магическими символами (не загромождено)
- Цвета: сланец (стены) + фиолетовый неон (магические символы)
- Минимализм: несколько предметов (стол, книги), много пустоты
- Настроение: уютно, но магически опасно

### 4. WORLD MAP
- Вид: вид сверху на мир, районы отмечены неоновыми точками
- Цвета: чёрный фон карты + неоновые маркеры (голубой, фиолетовый, оранжевый)
- Минимализм: много пустого пространства, только нужные локации
- Ощущение: огромный мир, маг — маленькая точка в нём

### 5. COMBAT
- Вид: маг слева, враг/манекен справа, магический взрыв посередине
- Цвета: тёмный фон + яркие неоновые эффекты заклинаний
- Минимализм: фокус на персонажах и магии, фон минимален
- Ощущение: напряжение, скорость, магическая сила

## ТЕХНИЧЕСКИЕ ПАРАМЕТРЫ ДЛЯ MIDJOURNEY

**Всегда добавляй в конец промпта:**
```
--ar 16:9  (для фонов, если не указано иное)
--niji 6   (для pixel art может работать, или --v 6)
--q 2      (качество, важно для пиксельности)
-no blur, realistic, photorealistic, soft edges
```

**Для спрайтов:**
```
--ar 1:1  (квадрат)
--bg      (для прозрачности)
```

**Для иконок:**
```
--ar 1:1
--s 750   (меньше креативности, больше точности)
```
