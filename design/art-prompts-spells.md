# Veyra Class Spells — Pixel Art Icon Prompts

**Document**: Art direction for all class spell icons (48x48px, pixel art style)
**Project**: Veyra Dark Fantasy RPG
**Date**: 2026-03-30
**Status**: DRAFT — prompts ready for generation

---

## Style Guidelines

All spell icons follow these rules:
- **Format**: 48×48 pixel art, centered composition
- **Background**: Transparent (png)
- **Palette**: Class-specific + muted dark fantasy tones
- **Detail**: Bold, readable at small size; 2-4 color layers per icon
- **Style**: Pixel art, dark fantasy RPG, inspired by Diablo 2 spell icons
- **Contrast**: High contrast against dark UI backgrounds
- **Glow**: Optional inner glow matching class color (not mandatory in prompt, let AI decide)

### Class Color Codes

| Class | Primary | Secondary | Accent |
