# Shelby Dark Fantasy — MVP Game Design Document (Sprint 1)

**Версия:** 1.0 | **Дата:** 2026-03-15
**Платформа:** Браузер (PC only), GitHub Pages
**Жанр:** Dark Fantasy RPG (casual, daily engagement)
**Целевая сессия:** 3-5 минут/день
**Монетизация MVP:** Нет. Проект для комьюнити разработчиков экосистемы Shelby. Потенциал: блокчейн-механики в будущих спринтах (NFT-снаряжение, токены).
**Хранение данных:** `localStorage` браузера (нет бэкенда).

---

## Контекст и цель

MVP должен дать ощущение прогрессии и причину возвращаться каждый день. Core loop:

```
[Вход] -> [Забрать Daily Login] -> [Карта] -> [Тренировочная площадка: 5 боёв]
  -> [Получить золото + XP + шанс лута] -> [Проверить инвентарь, экипировать]
  -> [Выход / ждать завтра]
```

---

## 1. Боевая система — Тренировочная площадка

### 1.1 Тип боя: полуавтоматический с выбором заклинания

Игрок выбирает 1 из 3 заклинаний. Маг проигрывает анимацию каста, манекен получает урон. Один клик за раунд — не автобой (есть осмысленный выбор), но и не экшн.

### 1.2 Правила

- Бой: **3 раунда**, в каждом выбираешь 1 из 3 заклинаний
- Манекен HP: **100**. Манекен не атакует
- Лимит: **5 боёв в день**. Сброс в 00:00 по локальному времени
- Неиспользованные бои не переносятся

### 1.3 Заклинания мага

| Заклинание | Тип | Мин. урон | Макс. урон | Средний |
|---|---|---|---|---|
| **Arcane Bolt** (Чародейский разряд) | Быстрый | 25 | 35 | 30 |
| **Verdant Flame** (Зелёное пламя) | Мощный | 20 | 60 | 40 |
| **Shadow Strike** (Теневой удар) | Средний | 30 | 40 | 35 |

**Анимации:**
- Arcane Bolt: синий светящийся шар летит в цель
- Verdant Flame: зелёное пламя окутывает манекен
- Shadow Strike: фиолетовая тень бьёт из-под земли

### 1.4 Формула урона

```
FinalDamage = floor( RandomInt(MinDmg, MaxDmg) * (1 + BonusPower / 100) )
```

`BonusPower` = сумма бонусов от экипированного снаряжения. Начинается с 0.

### 1.5 Вероятность провала при BonusPower = 0: ~15-20%

### 1.6 Награды за бой

| Результат | Золото | XP | Шанс Common-дропа |
|---|---|---|---|
| **Победа** | 10-15 (random) | 20 | 3% |
| **Провал** | 5-7 (random) | 10 | 0% |

**Бонус за серию (5 побед за день):** +25 золота, +50 XP

## 2. Daily Login — Ежедневные награды

### 2.1 Правила

- Цикл **7 дней**, затем рестарт
- Пропуск дня: прогресс **НЕ сбрасывается**
- Награда выдаётся автоматически при первом входе за день

### 2.2 Таблица наград

| День | Награда |
|---|---|
| 1 | 10 золота |
| 2 | 20 золота |
| 3 | 30 золота + 1 Эликсир маны* |
| 4 | 50 золота |
| 5 | 40 золота + 1 Осколок кристалла** |
| 6 | 75 золота |
| 7 | **Сундук тренировки** (гарантированный Common-предмет) |

\* Эликсир маны — предмет-заглушка с текстом "Пригодится позже..."
\** Осколок кристалла — заглушка для будущей валюты (потенциально блокчейн-токен)

## 4. Инвентарь и снаряжение

### 4.1 Слоты экипировки

| Слот | Стартовый предмет | BonusPower |
|---|---|---|
| **Посох** | Ученический посох | +0 |
| **Шляпа** | Потрёпанная остроконечная шляпа | +0 |
| **Накидка** | Выцветшая зелёная накидка | +0 |

### 4.2 Система редкостей

| Редкость | Цвет | BonusPower | В MVP? |
|---|---|---|---|
| **Common** | Белый | +3...+7 | Да |
| **Uncommon** | Зелёный | +8...+14 | Нет |
| **Rare** | Синий | +15...+24 | Нет |
| **Epic** | Фиолетовый | +25...+39 | Нет |
| **Legendary** | Оранжевый | +40...+60 | Нет |

### 4.3 Пул Common-предметов (MVP)

| Предмет | Слот | BonusPower | Описание |
|---|---|---|---|
| Дубовый посох | Посох | +3 | Крепкий и надёжный |
| Посох с аметистом | Посох | +5 | Фиолетовый камень слабо мерцает |
| Посох болотного огня | Посох | +7 | Навершие тлеет зелёным |
| Шляпа послушника | Шляпа | +3 | Простая, но целая |
| Шляпа звездочёта | Шляпа | +5 | Вышитые звёзды на полях |
| Шляпа ночного ветра | Шляпа | +7 | Лёгкая, будто сшита из тумана |
| Дорожная накидка | Накидка | +3 | Пахнет дымом костра |
| Накидка лесного стража | Накидка | +5 | Тёмно-зелёная, с капюшоном |
| Накидка сумерек | Накидка | +7 | Фиолетовые отблески на ткани |

## 5. Прогрессия персонажа

### 5.1 Формула XP

```
XP_для_уровня(N) = floor(100 * N^1.3)
```

### 5.2 Таблица уровней (1-10)

| Уровень | XP до след. | Дней фарма | Награда |
|---|---|---|---|
| 1→2 | 100 | ~1 | +5 золота |
| 2→3 | 246 | ~1 | +10 золота |
| 3→4 | 415 | ~2 | +15 золота |
| 4→5 | 604 | ~4 | +20 золота + Сундук тренировки |
| 5→6 | 812 | ~7 | +25 золота + Звание "Адепт" |
| 6→7 | 1036 | ~11 | +30 золота |
| 7→8 | 1275 | ~16 | +35 золота + Сундук тренировки |
| 8→9 | 1528 | ~22 | +40 золота |
