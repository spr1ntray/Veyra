# Блокчейн-экосистемы для Veyra: пересмотр с новыми критериями

**Дата исследования:** 2026-03-26 (обновление)
**Исследователь:** Researcher Agent
**Источники:** Экспертные знания до мая 2025. Веб-поиск был недоступен — данные по TVL/DAU/грантам требуют перепроверки через DeFiLlama, DappRadar, официальные сайты.

---

## Исключённые сети и причины

| Сеть | Причина исключения |
|---|---|
| **TON** | Платформа Telegram Mini Apps, не ПК-браузер. Исключён по критерию #2 |
| **Ronin** | Держится на угасающем Axie Infinity. Pixels — единственный второй проект. Экосистема де-факто полумёртвая за пределами 1-2 игр. Исключён по критерию #3 |
| **Ethereum L1** | Комиссии $1-50+ за транзакцию, нет игрового фокуса. Исключён из предыдущего анализа |
| **Aptos** | Незрелая GameFi-экосистема, Move — высокий порог входа, малая аудитория. Исключён из предыдущего анализа |

---

## TL;DR (Ключевые выводы)

- **Base (Coinbase L2)** — лучший баланс: живая сеть с огромным TVL, EVM/vanilla JS простота, мало GameFi конкурентов, Coinbase Smart Wallet для онбординга, растущие гранты.
- **ImmutableX** — остаётся сильным кандидатом для NFT-экономики: gasless, принудительные royalties, Passport. Но экосистема нишевая и аудитория ограничена.
- **Sui** — уникальная объектная модель для RPG-предметов, zkLogin, низкие комиссии. Но Move — порог входа, и vanilla JS интеграция сложнее.
- **Polygon** и **Arbitrum** — живые EVM-сети, но перегружены конкурентами в GameFi.
- **Solana** — мощная экосистема, но SDK заточен под React, контракты на Rust, для vanilla JS — значительно сложнее.

## Детальный анализ: 8 сетей

### 1. Solana

**Активность сети:**
- DAU: ~1-2M active wallets (одна из самых активных сетей)
- TVL: ~$5-8B (данные волатильны, проверь DeFiLlama)
- Объём транзакций: десятки миллионов в день
- Статус: живая, очень активная сеть

**GameFi проекты (конкуренция):**
- Высокая конкуренция: Star Atlas, Aurory, Genopets, Stepn, Honeyland, Photo Finish, BR1 и десятки других
- Magic Eden — доминирующий маркетплейс с развитым gaming-разделом
- Оценка: **перегруженный рынок**, 50+ активных GameFi проектов
- Для нового инди-RPG выделиться будет сложно

**Гранты:**
- Solana Foundation Grants: до $250K
- Gaming-направление: до $150K отдельно
- Требования: рабочий MVP, open-source приветствуется
- Процесс: средней сложности

**Интеграция wallet для ПК браузера (vanilla JS):**
- @solana/web3.js — работает в браузере
- Phantom, Solflare — ПК-браузерные кошельки, популярные
- **Проблема:** wallet-adapter SDK заточен под React. Для vanilla JS нужно писать обёртки вручную — подключение Phantom через window.solana возможно, но нестандартно
- Контракты: Rust (Anchor framework) — высокий порог входа
- Оценка интеграции: **6/10** (браузерный кошелёк есть, но SDK не для vanilla JS)

**Комиссии:**
- ~$0.00025 за транзакцию + rent ~$0.002 за создание аккаунта
- Compressed NFTs: минтинг за доли цента
- Оценка: отличные

**Вердикт:** Живая мощная сеть, но перегружена GameFi проектами и техническая интеграция для vanilla JS заметно сложнее EVM-сетей. Rust для контрактов — серьёзный барьер.

### 2. ImmutableX / Immutable zkEVM

**Активность сети:**
- DAU: ~100-300K (почти все — геймеры)
- TVL: относительно невысокий (~$100-200M), сеть ориентирована на NFT-объёмы, не DeFi
- Объём NFT-торговли: стабильный, но снизился с пиков 2021-2022
- Статус: живая, но нишевая. Экосистема целиком зависит от gaming

**GameFi проекты (конкуренция):**
- Gods Unchained, Illuvium, Guild of Guardians, Cross The Ages, Shardbound, MetalCore
- ~20-40 активных игровых проектов
- Оценка: **умеренная конкуренция**. Проекты есть, но рынок не перегрет. Место для качественной RPG найдётся
- Уникальный плюс: вся аудитория — геймеры, не DeFi-трейдеры

**Гранты:**
- Immutable Developer Grants: до $500K
- $500M экосистемный фонд (с 2022)
- Immutable Ventures: прямые инвестиции
- Игровой фокус — обязателен (это плюс для Veyra)
- Процесс: средней сложности, но gaming-проекты в приоритете

**Интеграция wallet для ПК браузера (vanilla JS):**
- Immutable Passport: social login (email/Google) — без кошелька вообще. Идеально для онбординга
- zkEVM: EVM-совместимый, можно подключить MetaMask стандартно
- Immutable SDK (TypeScript): работает в браузере, можно собрать bundle для vanilla JS
- REST API как fallback — можно обойтись без SDK вообще
- Оценка интеграции: **7/10** (Passport снижает барьер, но SDK ориентирован на TS)

**Комиссии:**
- IMX L2: $0 gas для NFT-операций (minting, transfer, trade)
- zkEVM: ~$0.01-0.05
- Для RPG с частыми операциями — идеально

**Вердикт:** Остаётся сильным кандидатом. Gasless NFT + принудительные royalties + Passport — уникальная комбинация для GameFi. Риск: нишевая сеть, и если IMX как компания теряет позиции, экосистема может сжаться.

### 3. Polygon (PoS / zkEVM)

**Активность сети:**
- DAU: ~300-500K active wallets
- TVL: ~$1-2B (PoS) + растущий zkEVM
- Объём транзакций: миллионы в день
- Статус: живая, стабильная сеть с широкой экосистемой

**GameFi проекты (конкуренция):**
- Aavegotchi, Sunflower Land, Zed Run, Sandbox (частично), Skyweaver, множество мелких проектов
- 100+ GameFi проектов (по данным DappRadar)
- Оценка: **перегруженный рынок**. Polygon стал "default L2" для многих, и GameFi-ниша забита

**Гранты:**
- Polygon Village Grants: до $75K
- Polygon Ecosystem Fund: $100M+
- Требования: рабочий прототип, roadmap
- Процесс: средней сложности, но не специфичен для gaming

**Интеграция wallet для ПК браузера (vanilla JS):**
- EVM-совместимая: ethers.js / web3.js / viem — все работают в vanilla JS
- MetaMask — стандартный, добавить сеть тривиально
- Документация отличная, множество туториалов
- Оценка интеграции: **9/10** — одна из самых простых

**Комиссии:**
- $0.001-0.01 за транзакцию (PoS)
- Приемлемо для игровых механик

**Вердикт:** Технически простейшая интеграция, но рынок GameFi перегрет. Трудно выделиться среди сотни проектов. Гранты небольшие.

### 4. Sui

**Активность сети:**
- DAU: ~200-500K (быстро рос в 2024-2025)
- TVL: ~$500M-1B (рос агрессивно)
- Объём транзакций: десятки миллионов в день (дешёвые транзакции стимулируют объём)
- Статус: живая, активно растущая

**GameFi проекты (конкуренция):**
- Panzerdogs, Abyss World, Run Legends, Worlds Beyond — немного
- ~10-20 активных GameFi проектов
- Оценка: **низкая конкуренция**. Экосистема растёт, но GameFi-ниша ещё не забита. Отличное окно входа

**Гранты:**
- Sui Foundation Grants: до $500K
- Sui Gaming Initiative: активное привлечение игровых студий
- Builder House программы
- Требования: использование Move, прототип
- Процесс: средней сложности, gaming — в приоритете

**Интеграция wallet для ПК браузера (vanilla JS):**
- @mysten/sui.js — работает в браузере
- Sui Wallet, Ethos Wallet, Suiet — ПК-браузерные расширения
- zkLogin: вход через Google/Apple без кошелька — снижает барьер
- **Проблема:** Move для контрактов — нужно изучать с нуля. SDK ориентирован на TypeScript
- Документация улучшается, но уступает EVM-экосистеме
- Оценка интеграции: **5/10** (zkLogin — плюс, но Move и TS-ориентированный SDK — минусы)

**Комиссии:**
- $0.001-0.01 за транзакцию
- Очень низкие

**Уникальные преимущества для RPG:**
- Объектная модель: каждый NFT — мутабельный объект. Предмет может "расти", получать зачарования on-chain без пересоздания
- Kiosk стандарт: принудительные royalties с creator-контролем
- Динамические NFT нативно — идеально для RPG с эволюционирующими предметами

**Вердикт:** Уникальная технология для RPG, низкая конкуренция, крупные гранты. Но Move — серьёзный порог входа, и для vanilla JS интеграция сложнее EVM.

### 5. Avalanche (C-Chain / Subnets)

**Активность сети:**
- DAU: ~100-200K
- TVL: ~$1-2B (в основном DeFi)
- Статус: живая, стабильная, но не растёт так быстро, как Sui или Base

**GameFi проекты (конкуренция):**
- DeFi Kingdoms (мигрировал на Subnet, но популярность упала), Shrapnel, Pulsar
- ~15-30 GameFi проектов
- Оценка: **умеренная конкуренция**. Есть место, но экосистема не фокусируется на gaming

**Гранты:**
- Blizzard Fund: $200M+
- Avalanche Multiverse: $290M для Subnets
- До $500K для значимых проектов
- Subnets — уникальная возможность создать свою цепочку

**Интеграция wallet для ПК браузера (vanilla JS):**
- EVM-совместимая: ethers.js / web3.js
- Core Wallet (нативный) + MetaMask
- Оценка интеграции: **8/10** — стандартная EVM-интеграция

**Комиссии:**
- C-Chain: $0.01-0.1
- Subnet: настраиваемо, вплоть до $0

**Вердикт:** Subnets — мощная концепция (своя цепочка для игры), но для инди-RPG на старте это избыточно. Экосистема не gaming-focused.

### 6. Base (Coinbase L2)

**Активность сети:**
- DAU: ~1-2M active wallets (одна из самых быстрорастущих L2 в 2024-2025)
- TVL: ~$3-7B (агрессивный рост, одна из крупнейших L2)
- Объём транзакций: десятки миллионов в день
- Статус: **очень живая**, поддержка Coinbase, стремительный рост

**GameFi проекты (конкуренция):**
- Относительно немного GameFi-проектов: основная активность — DeFi, social (Farcaster/friend.tech), NFT
- ~10-20 GameFi проектов
- Оценка: **низкая конкуренция в GameFi**. Огромная аудитория, но gaming-ниша практически пустая. Отличное окно входа

**Гранты:**
- Base Ecosystem Fund: Coinbase Ventures инвестирует в экосистему
- Base Builder Grants: программы для разработчиков (суммы варьируются, до $50-100K)
- Onchainsummer и другие инициативы
- Не такие крупные как у Sui/IMX для gaming, но Coinbase backing даёт вес
- Требования: строить на Base, EVM

**Интеграция wallet для ПК браузера (vanilla JS):**
- EVM-совместимая: ethers.js / web3.js / viem — полная совместимость
- MetaMask: добавить Base как сеть — тривиально
- **Coinbase Smart Wallet**: passkey-based, gasless onboarding без расширения — пользователь создаёт кошелёк в 2 клика без установки MetaMask
- OnchainKit (Coinbase): набор компонентов, но есть и vanilla JS примеры
- Оценка интеграции: **9/10** — EVM простота + Smart Wallet для онбординга

**Комиссии:**
- ~$0.001-0.01 за транзакцию (после EIP-4844/blobs — значительно снизились)
- Coinbase Smart Wallet может спонсировать газ для пользователей (paymaster)

**Уникальные преимущества:**
- Coinbase backing: 100M+ верифицированных пользователей Coinbase, которые могут легко перейти на Base
- Smart Wallet: onboarding без MetaMask — конкурентное преимущество наравне с Immutable Passport
- Farcaster-экосистема: social layer для вирусности
- Сеть растёт, а GameFi-ниша ещё не занята — можно стать одним из первых заметных gaming-проектов

**Вердикт:** **Лучший баланс критериев.** Живая растущая сеть, простейшая интеграция (EVM + vanilla JS), Smart Wallet для онбординга, минимум GameFi-конкурентов на огромном рынке. Гранты поменьше, чем у IMX/Sui, но Coinbase-экосистема компенсирует.

### 7. Arbitrum

**Активность сети:**
- DAU: ~500K-1M
- TVL: ~$3-5B (одна из крупнейших L2)
- Объём транзакций: миллионы в день
- Статус: живая, зрелая, стабильная

**GameFi проекты (конкуренция):**
- Treasure DAO (The Beacon, Smolverse), Pirate Nation, TreasureDAO marketplace
- ~30-50 GameFi проектов через Treasure ecosystem
- Оценка: **умеренно-высокая конкуренция**. Treasure уже создала gaming-хаб на Arbitrum, и новому проекту придётся конкурировать внутри этой экосистемы

**Гранты:**
- Arbitrum Foundation Grants: через DAO governance (STIP, LTIP программы)
- Gaming-специфичных грантов мало — больше DeFi-фокус
- Суммы: варьируются, процесс через DAO — медленный и непредсказуемый
- Treasure DAO имеет свои гранты для игр в своей экосистеме

**Интеграция wallet для ПК браузера (vanilla JS):**
- EVM-совместимая: ethers.js / web3.js
- MetaMask — стандартно
- Оценка интеграции: **9/10** — стандартный EVM

**Комиссии:**
- ~$0.001-0.05 за транзакцию
- Низкие после EIP-4844

**Вердикт:** Живая сеть, простая интеграция, но GameFi-ниша уже занята Treasure DAO. Гранты через DAO — медленный процесс. Не лучший выбор для нового инди-RPG.

### 8. Starknet

**Активность сети:**
- DAU: ~50-150K (меньше, чем у других L2)
- TVL: ~$200-500M
- Объём транзакций: растёт, но отстаёт от Arbitrum/Base
- Статус: живая, но молодая. Технологически интересная (ZK-rollup), но экосистема ещё формируется

**GameFi проекты (конкуренция):**
- Realms (on-chain strategy), Influence, Topology — несколько проектов
- Cartridge.gg — gaming framework для Starknet
- Dojo — ECS engine для on-chain games
- ~5-15 GameFi проектов
- Оценка: **очень низкая конкуренция**. Но и аудитория маленькая

**Гранты:**
- StarkNet Foundation Grants: существуют
- Размер: до $250K
- Фокус: инфраструктура и инновации, gaming не в приоритете

**Интеграция wallet для ПК браузера (vanilla JS):**
- starknet.js — работает в браузере
- ArgentX, Braavos — ПК-браузерные кошельки
- **Проблема:** контракты на Cairo — нишевый язык, высокий порог входа
- Account abstraction нативно — интересно для UX
- Оценка интеграции: **4/10** (Cairo — серьёзный барьер, малая экосистема инструментов)

**Комиссии:**
- ~$0.01-0.1 (снижаются с развитием)
- Приемлемые

**Вердикт:** Технологически интересна (fully on-chain gaming с Dojo), но Cairo — высочайший порог входа, малая аудитория, gaming не в приоритете фонда. Для инди-RPG на vanilla JS — не подходит.

## Рекомендация: ТОП-3 с обоснованием

### 1. Base (Coinbase L2) — рекомендация номер один

**Почему:**
- Самая быстрорастущая L2 с огромной живой аудиторией (1-2M DAU) и поддержкой Coinbase (100M+ пользователей)
- GameFi-ниша практически пустая — можно стать одним из первых заметных gaming-проектов на крупной сети
- EVM-совместимая — ethers.js в vanilla JS, без фреймворков. MetaMask из коробки
- Coinbase Smart Wallet — passkey-based onboarding без установки расширения, gasless через paymaster. Конкурирует с Immutable Passport по удобству
- Solidity для контрактов — самый большой пул разработчиков и документации
- Комиссии копеечные после EIP-4844

**Риски:**
- Гранты для gaming не такие крупные, как у IMX/Sui (до $100K vs $500K)
- Нет встроенного маркетплейса с принудительными royalties — нужно строить самим или использовать существующие (OpenSea на Base)
- Нет native gaming infra (в отличие от IMX Passport/Orderbook)

**Для Veyra конкретно:**
- Dark fantasy RPG на Base выделится моментально — сеть забита DeFi и social, но gaming почти нет
- Smart Wallet позволит онбордить не-крипто геймеров через passkey (Face ID / отпечаток)
- Контракты на Solidity — максимально доступный навык на рынке

**Контакт:** https://base.org/grants

---

### 2. ImmutableX / Immutable zkEVM — лучший для NFT-экономики

**Почему:**
- Единственная сеть, полностью заточенная под gaming. Все инструменты (Passport, Orderbook, gasless minting) — для игр
- Gasless NFT — игроки не платят за минт/трейд. Для RPG с экипировкой это критично
- Принудительные royalties — гарантированный доход с вторичных продаж предметов
- Immutable Passport — social login через email, без кошелька. Работает в ПК-браузере
- Гранты до $500K с прямым gaming-фокусом

**Риски:**
- Нишевая сеть: если убрать gaming, TVL и активность невысокие
- Зависимость от одной компании (Immutable) — если они потеряют фокус, экосистема пострадает
- SDK заточен под TypeScript, для vanilla JS потребуется бандлинг

**Для Veyra конкретно:**
- Посохи, доспехи, свитки — всё как gasless NFT с royalties
- Global Orderbook — листинг на всех IMX маркетплейсах одновременно
- Passport снижает барьер входа для не-крипто аудитории

**Контакт:** https://www.immutable.com/developers

---

### 3. Sui — если динамические NFT критичны для геймплея

**Почему:**
- Объектная модель: предмет может эволюционировать on-chain (зачарования, прокачка, износ) без пересоздания токена. Для RPG — уникальное преимущество
- Kiosk стандарт с принудительными royalties
- zkLogin — вход через Google/Apple без крипто-кошелька
- Гранты до $500K, gaming — в приоритете фонда
- GameFi-ниша не перегружена (10-20 проектов)
- Сеть активно растёт

**Риски:**
- Move — нужно изучать новый язык с нуля. Пул разработчиков маленький
- SDK ориентирован на TypeScript. Для vanilla JS — дополнительная работа
- Экосистема инструментов менее зрелая, чем EVM

**Для Veyra конкретно:**
- Меч с +5 к атаке, получивший зачарование огня — обновляется on-chain как мутабельный объект, а не пересоздаётся
- Если геймплей Veyra завязан на эволюцию предметов, Sui даёт техническое превосходство

**Контакт:** https://sui.io/grants

## Стратегическая рекомендация

Оптимальная стратегия для Veyra — **Base как основная сеть**, с возможным добавлением ImmutableX для NFT-маркетплейса позже:

| Фаза | Сеть | Что делаем |
|---|---|---|
| MVP / Launch | **Base** | Основная игра, Smart Wallet onboarding, базовые NFT предметы, Solidity контракты |
| Масштабирование | **Base + ImmutableX** | Маркетплейс предметов на IMX (gasless trading, royalties), bridge между сетями |
| Альтернатива | **Sui** | Только если динамические NFT станут ключевой механикой и команда готова осваивать Move |

**Почему Base первым:**
1. Минимальный time-to-market: EVM + Solidity + vanilla JS + MetaMask = можно начать завтра
2. Максимальная аудитория живой сети при минимуме GameFi-конкурентов
3. Smart Wallet решает проблему онбординга без специального gaming SDK
4. Если потребуется NFT-маркетплейс с royalties — можно добавить IMX позже через bridge

## Важные предупреждения

- Все данные по TVL/DAU актуальны до мая 2025. Для принятия решения рекомендуется проверить актуальные метрики на [DeFiLlama](https://defillama.com), [DappRadar](https://dappradar.com), [L2Beat](https://l2beat.com)
- Грантовые программы меняются — перед подачей проверь условия на официальных сайтах
- Royalties enforcement — политики маркетплейсов могут измениться
- Base не имеет своего токена (по состоянию на май 2025) — это может измениться и повлиять на экономику
