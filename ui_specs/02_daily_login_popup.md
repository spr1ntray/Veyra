# DailyLoginPopup — Ежедневная награда

## Описание
Модальное окно, появляющееся автоматически при первом входе за новый день. Показывает 7-дневный цикл наград, текущий день подсвечен, кнопка "Забрать" выдаёт награду и закрывает попап.

---

## Допущения
- [ASSUMPTION] Попап появляется поверх WorldMap (первый вход) или поверх LoadingScreen transition.
- [ASSUMPTION] Нажатие вне попапа НЕ закрывает его — игрок обязан нажать "Забрать".
- [ASSUMPTION] После клика "Забрать" награда мгновенно зачисляется в localStorage; анимация выдачи — 1.5s, затем попап исчезает.

## Состояния и поведение

### Появление попапа
- Overlay: fade-in opacity 0 → 0.65 за 0.3s.
- `popup_container`: scale 0.85 → 1.0 + opacity 0 → 1 за 0.35s, ease-out.
- Задержка между overlay и popup: 0.05s.

### Default (попап открыт, награда ещё не забрана)
- `claim_button`: bg `#c9a84c`, text `#0a0e27`, cursor pointer.

### Hover (claim_button)
- Bg: `#d4b55c` (светлее на 8%).
- Box-shadow: `0 0 20px #c9a84c66`.
- Scale: 1.02.
- Transition: 0.15s.

### Active (claim_button при клике)
- Scale: 0.96.
- Bg: `#a8893d`.

### После клика "Забрать"
1. Кнопка переходит в `disabled`: bg `#4a3a2a`, text `#8a7a6a`, cursor not-allowed.
2. Текущая ячейка `day_cell[current]`: анимация золотых частиц (8–12 частиц разлетаются из центра ячейки за 0.8s, fade-out).
3. Через 0.5s: ячейка переходит в состояние `claimed` (галочка появляется со scale 0 → 1.2 → 1.0 за 0.3s).
4. Через 1.5s после клика: попап закрывается — `popup_container` scale 1.0 → 0.9 + opacity → 0 за 0.3s, затем overlay fade-out.

## Notes for Developer

- `overlay_backdrop` блокирует все pointer-events ниже попапа.
- Награда зачисляется в localStorage сразу при клике (до анимации), чтобы не потерять при случайном закрытии вкладки.
- Если `currentDay === 7` и следующий клик — новый цикл, `currentDay` сбрасывается в 1 после анимации.
- `cyclesCompleted` инкрементируется при переходе из дня 7 в день 1.
- `day_grid` — горизонтальный flex, nowrap, gap 8px, выровнен по центру. На 1280px все 7 ячеек умещаются без скролла.

## Notes for Art Director

- Рамка попапа: орнаментальные углы в стиле средневековой рукописи — тонкие линии с ромбами и засечками. Цвет рамки `#3a2a5a`, уголковые акценты `#c9a84c`.
- Ячейки дней: не карточки с тенью — плоские, строгие прямоугольники с тонкой рамкой. Текущий день выделяется только золотой рамкой и свечением, не объёмом.
- Частицы при получении награды: маленькие шестиугольные золотые осколки или звёздочки. Не конфетти — только золото и изумруд.
- Фон попапа `#1a0a2e` немного темнее основного фона карты, чтобы выглядеть как отдельный слой.
